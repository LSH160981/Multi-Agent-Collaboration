# 发布说明草案

## 项目名称

Multi-Agent-Collaboration

## 一句话说明

OpenClaw 原生多会话多 Agent 协作系统，把复杂任务升级为主Agent统筹、动态招聘、双组竞争、审核评分与巡检恢复的协作流程。

## 核心特性

- Agent = OpenClaw session
- `/mac` 显式触发多 Agent 流程
- 主Agent是唯一用户出口
- 动态 specialist 招聘
- A/B 双组竞争
- 审核Agent 三维评分
- 检查Agent 巡检与恢复
- JSON 协议 / 样例链 / 状态 schema

## 适合人群

- 需要把复杂任务从单 Agent 提升为多 Agent 协作的人
- 想把 OpenClaw 用成“AI 公司”工作流的人
- 想研究多 Agent 协议、评分、恢复与演化机制的人

## 当前状态

当前为高完成度原型仓库，已具备初始化、自检、编组、样例、schema、文档与演示手册。
