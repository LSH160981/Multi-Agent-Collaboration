# 目录结构设计

```text
Multi-Agent-Collaboration/
├── README.md
├── CHANGELOG.md
├── VERSION
├── skills/
│   └── Multi-Agent-Collaboration/
├── agents/
│   ├── 主Agent/
│   ├── 审核Agent/
│   ├── 检查Agent/
│   └── AgentPool/
├── templates/
│   ├── agent/
│   └── shared/
├── research/
│   ├── sources/
│   └── github/
└── docs/
    ├── 平台级接管设计.md
    └── 目录结构设计.md
```

## 设计原则

- skill 本体放 `skills/`
- 长期角色骨架放 `agents/`
- 动态生成所需模板放 `templates/`
- 学习材料放 `research/`
- 系统设计说明放 `docs/`
