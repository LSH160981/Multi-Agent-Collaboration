# 演示跑通手册

这份手册用于把 Multi-Agent-Collaboration 从“仓库结构”跑成“能看懂的系统”。

## 1. 安装 skill

把 `skills/Multi-Agent-Collaboration/` 放到 OpenClaw 工作区的 `skills/` 下，或共享技能目录下。

## 2. 初始化目录

```bash
./scripts/init-mac-system.sh
```

## 3. 自检

```bash
./scripts/install-selfcheck.sh
```

## 4. 生成 specialist（示例）

```bash
./scripts/generate-agent.sh ~/.openclaw/workspace "A组-前端专家-Frontend" "前端专家"
./scripts/generate-agent.sh ~/.openclaw/workspace "B组-检索专家-Research" "检索专家"
```

## 5. 让系统处理一个演示任务

示例：

```text
/mac 调研最近 30 天值得关注的 OpenClaw 多Agent 项目，提炼协同架构优点并给出改进建议。
```

## 6. 观察什么

- `examples/json/`：看消息流应该长什么样
- `examples/task-board-example.md`：看任务看板
- `examples/review-scorecard-example.md`：看评分卡
- `examples/log-example.md`：看日志格式
- `examples/queue-example.json`：看队列状态

## 7. 安装后建议必做测试

1. Agent 两两握手测试
2. 静默任务测试
3. 恢复测试

## 8. 目标

跑通这套流程后，你应该能清楚理解：

- 主Agent 如何统筹
- AgentPool 如何招聘与编组
- specialist 如何接单
- 审核Agent 如何评分
- 检查Agent 如何巡检
- 主Agent 如何最终向用户输出
