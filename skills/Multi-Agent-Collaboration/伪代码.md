# Multi-Agent-Collaboration 伪代码（中文）

## 1. 总控流程伪代码

```text
函数 处理用户任务(用户输入):
    主Agent = 激活主Agent()
    如果 用户输入缺少关键细节:
        主Agent.向用户提问(缺失信息)
        结束

    任务 = 主Agent.创建总任务(用户输入)
    AgentPool = 激活AgentPool()
    审核Agent = 激活审核Agent()
    检查Agent = 激活检查Agent()

    能力需求 = 主Agent.分析所需能力(任务)
    团队A, 团队B = AgentPool.按能力招聘两组(能力需求)

    主Agent.向团队A组长派单(任务)
    主Agent.向团队B组长派单(任务)

    并行执行:
        团队A.开始工作()
        团队B.开始工作()
        检查Agent.周期巡检(团队A, 团队B)

    当 两组都提交结果后:
        评分A = 审核Agent.审核并评分(团队A结果)
        评分B = 审核Agent.审核并评分(团队B结果)

        如果 评分A 和 评分B 都不达标:
            AgentPool.淘汰低分Agent或整组重建()
            返回 上一个执行阶段 重新做

        最优结果 = 审核Agent.选出更优方案(评分A, 评分B)
        主Agent.汇总去重(最优结果, 历史已发送信息)
        主Agent.向用户输出(最终答案)
```

## 2. 动态招聘伪代码

```text
函数 按能力招聘两组(能力需求):
    候选Agent列表 = 读取现有Agent能力库()

    对于 每一种能力 in 能力需求:
        如果 候选Agent列表中存在可复用Agent:
            复用它
        否则:
            新建Agent(指定角色, 能力边界, 工具权限, 记忆目录, 日志目录, 队列目录)

    把Agent分成A组和B组
    每组指定一名组长
    返回 A组, B组
```

## 3. Agent 派单伪代码

```text
函数 派单(发送方, 接收方, 任务):
    消息 = {
        type: "task_assign",
        task_id: 任务.id,
        from: 发送方,
        to: 接收方,
        goal: 任务.goal,
        constraints: 任务.constraints,
        inputs: 任务.inputs,
        deliverables: 任务.deliverables,
        status: "assigned"
    }

    写入发送方日志(消息)
    写入接收方队列(消息)
    发送JSON消息(接收方, 消息)
```

## 4. 审核评分伪代码

```text
函数 审核并评分(结果包):
    格式分 = Reviewer.检查格式完整性(结果包)
    质量分 = Judge.检查内容质量(结果包)
    指标分 = Metrics.检查可验证性与稳定性(结果包)

    总分 = 加权计算(格式分, 质量分, 指标分)

    如果 总分 < 阈值:
        返回 {状态: "reject", 原因: 详细问题, 总分: 总分}
    否则:
        返回 {状态: "pass", 优点: 提炼亮点, 总分: 总分}
```

## 5. 巡检恢复伪代码

```text
函数 周期巡检(多个Agent组):
    每隔1分钟执行一次:
        对于 每个Agent:
            最近动作 = 读取日志和任务队列()
            如果 长时间无动作 或 没有产出:
                发送激活消息(Agent)
                执行 retry 或重派单

            如果 Agent持续失败:
                读取其上下文、日志、产物
                判断是否重建
                如果 需要重建:
                    创建替代Agent
                    继承必要任务上下文
                    恢复执行
```

## 6. 用户消息去重伪代码

```text
函数 汇总去重(候选结果, 历史消息):
    删除重复观点
    删除无效中间状态
    合并异步回执
    删除内部术语噪音
    保留最新有效结论
    生成用户可读版本
    返回 用户可读版本
```
