# 自动自学习方案

## 目标

让本 skill 周期性学习新的多 Agent 协作方案，并把有价值的内容沉淀为自己的改进素材。

## 学习来源

1. ClawHub 上的相关 skill
2. GitHub 上的多 Agent / orchestration / collaboration 项目
3. 博客文章、案例复盘、架构分享
4. OpenClaw 官方文档更新

## 每次学习提取什么

- 角色设计
- 通信协议
- 任务状态机
- 审核评分机制
- 恢复与巡检机制
- 自动化测试方法
- 伪代码与流程图

## 学习结果落地

- 写入 `research/`
- 更新 `SKILL.md`
- 更新 `伪代码.md`
- 更新 `逻辑执行流程.md`
- 必要时更新 agent 模板与测试模板

## 自学习边界

允许：

- 搜索资料
- 阅读资料
- 提炼优缺点
- 改进本仓库文档与模板

禁止：

- 未经确认擅自修改用户私有配置
- 未经确认擅自升级系统
- 未经确认向外部发送信息

## 建议执行频率

- 每周 1-3 次小规模学习
- 每月 1 次结构化复盘
