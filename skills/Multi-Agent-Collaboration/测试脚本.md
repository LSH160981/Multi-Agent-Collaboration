# 测试脚本

## 测试一：Agent 两两握手

目标：验证所有 Agent 之间的通信能力。

### 发送模板

```text
我是{agent_name}，你是谁？
```

### 回复模板

```text
我是{agent_name}，我的能力是{abilities}。
```

### 验收标准

- 每个 Agent 都能收到其他 Agent 的握手
- 每个 Agent 都能回自己的身份与能力
- 日志和队列中能看到对应记录

## 测试二：静默任务测试

任务：

```text
搜索 GitHub 最近 7 天 star 涨得最快的 10 个项目，总结共同特点。
```

### 验收标准

- 主Agent 能正确拆解任务
- AgentPool 能按需招聘
- 两组团队都能完成调研
- 审核Agent 能给出评分
- 检查Agent 能识别卡住状态
- 主Agent 最终输出结果去重且清晰

## 测试三：恢复测试

模拟：

- specialist 中断
- lead 中断
- review 中断
- inspection 中断

### 验收标准

- 能从日志和队列恢复
- 不会重复大量返工
- 不会重复对用户发相同信息
