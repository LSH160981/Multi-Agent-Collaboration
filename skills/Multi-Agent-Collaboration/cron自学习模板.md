# cron 自学习模板

用于周期性触发本 skill 的自学习行为。

## 目标

定期搜索新的多 Agent 协作 skill、文章、开源项目，提炼可借鉴内容。

## 建议任务文案

```text
这是一次 Multi-Agent-Collaboration 的周期自学习任务。
请搜索最近出现的 OpenClaw / multi-agent / orchestration / collaboration 相关 skill、项目、文章。
重点提取：
1. 角色设计
2. JSON / A2A 通信方式
3. 审核评分与淘汰机制
4. 巡检恢复机制
5. 伪代码与执行流程
把结果写入 research/ ，如有必要更新 skill 文档与模板。
commit message 使用中文。
不要修改用户私有配置，不要外发消息。
```

## 建议频率

- 每周 1 次：轻量搜索 + 提炼
- 每月 1 次：结构化复盘 + 文档更新

## 结果要求

- 有来源
- 有优点提取
- 有对本 skill 的可用改进建议
- 避免无意义重复记录
