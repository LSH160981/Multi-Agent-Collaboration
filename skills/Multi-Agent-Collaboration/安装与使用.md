# 安装与使用

> 说明：本文件保留为历史手稿的简化导航版。当前以 `SKILL.md` + `references/` 作为主规范。

## 安装

把以下目录放入 OpenClaw 工作区的 `skills/` 目录：

- `skills/Multi-Agent-Collaboration/`
- `skills/mac/`（推荐，一并提供 `/mac` 命令桥）

如有需要，重新打开 session 或重启 gateway 让 skills 刷新。

## 使用入口

### 1. 复杂任务自然触发

当任务明显需要拆解、并行执行、审核或恢复能力时，可由主 Agent 默认采用本 skill 的方法论。

### 2. `/mac` 显式触发

```text
/mac 帮我做一个多步骤调研，并输出结构化报告
```

### 3. 点名 skill

```text
使用 Multi-Agent-Collaboration skill 完成这个复杂任务
```

## 核心约束

- 只有主 Agent 可以联系用户
- 非主 Agent 只能通过结构化消息、日志和产物回报
- 用户可见输出前必须去重、合并、降噪
