# 权重系统实例

## Agent 示例

### A组-前端专家-Frontend

- 审核通过率：91
- 平均审核分：88
- 时效分：82
- 稳定性分：90
- 协作分：85

计算：

```text
综合表现分 = 0.35*91 + 0.30*88 + 0.15*82 + 0.10*90 + 0.10*85
           = 31.85 + 26.4 + 12.3 + 9 + 8.5
           = 88.05
```

结论：

- 提升权重
- 下一轮优先复用

## Agent 示例

### B组-检索专家-Research

- 平均审核分：63
- 时效分：61
- 稳定性分：55
- 协作分：60

计算：

```text
综合表现分 = 0.35*58 + 0.30*63 + 0.15*61 + 0.10*55 + 0.10*60
           = 20.3 + 18.9 + 9.15 + 5.5 + 6

```

- 建议淘汰或重建
