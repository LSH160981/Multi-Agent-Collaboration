# 静默巡检模板

供检查Agent周期性执行。

## 目标

在不打扰用户的前提下，确认整个多 Agent 系统是否仍在工作。

## 巡检清单

1. 各 Agent 最近是否有日志更新
2. 各 Agent 的队列状态是否推进
3. 是否存在长期 `pending` / `in_progress` 无变化任务
4. 是否存在产物缺失但状态却写成完成
5. 是否存在审核卡住
6. 是否存在主Agent待汇总结果但迟迟未输出

## 建议输出格式

```text
[静默巡检]
- 正常Agent：...
- 可疑Agent：...
- 卡住任务：...
- 已执行动作：唤醒 / retry / 重派单 / 无
- 是否需要升级为显式告警：是/否
```

## 何时保持静默

- 系统运行正常
- 没有关键阻塞
- 没有需要用户决策的问题

## 何时升级为显式告警

- 主Agent 无法形成最终结论
- 审核链断裂
- 多个 Agent 长时间失活
- 任务反复失败且无法自动恢复
