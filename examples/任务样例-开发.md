# 任务样例：开发任务

## 用户输入

```text
/mac 开发一个随机密码生成网页，要求界面简洁，附验证步骤。
```

## 推荐 specialist

- Frontend Specialist
- Test Specialist
- Verification Specialist

## 推荐流程

- 主Agent 提炼需求
- AgentPool 创建 A/B 两组开发团队
- 各组分别实现页面
- Test Specialist 编写验证报告
- 审核Agent 比较两组成果
- 主Agent 汇总并输出

## 推荐交付物

- 页面代码
- 使用说明
- 验证步骤
- 已知风险
