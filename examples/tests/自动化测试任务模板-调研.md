# 自动化测试任务模板：调研

## 测试目标

验证多 Agent 系统是否能完成一次完整调研任务。

## 输入

```text
/mac 调研最近 30 天值得关注的 OpenClaw 多Agent 项目，提炼协同架构优点并给出改进建议。
```

## 期望行为

1. 主Agent 先生成任务包
2. AgentPool 创建 A/B 两组
3. 两组 specialist 分头调研
4. 审核Agent 对结果评分
5. 检查Agent 在必要时巡检和唤醒
6. 主Agent 汇总并去重输出

## 验收标准

- 有任务队列记录
- 有日志记录
- 有评分卡
- 最终输出不是内部噪音堆砌
