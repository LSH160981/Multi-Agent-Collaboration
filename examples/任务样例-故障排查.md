# 任务样例：故障排查任务

## 用户输入

```text
/mac 排查某个 OpenClaw 多Agent 协作任务经常卡死的原因，并给出修复和预防方案。
```

## 推荐 specialist

- Log Analysis Specialist
- Recovery Specialist
- Verification Specialist

## 推荐流程

- 主Agent 确认症状与范围
- AgentPool 创建 A/B 两组排障团队
- 一组偏日志分析，一组偏恢复与复现
- 审核Agent 对比两组结论
- 检查Agent 形成系统预防建议
- 主Agent 给用户最终结论

## 推荐交付物

- 根因分析
- 修复方案
- 验证步骤
- 预防策略
